动态库静态库(.so/.a)编译

  1. 编译环境搭建
    • CMake/Android Studio/Xcode
  2. 编译选项设定
    • 支持的cpu 架构
    • iOS(armv7 i386 x86_64 arm64)
    • [Android(‘x86’, ‘armeabi’, ‘armeabi-v7a’)
  3. SSL/TLS 加密编译以及设置
    • 打开编译选项 增加OPENSSL 宏 ;
    • 下载openssl 源码到本地电脑目录 ;
    • Xcode 或 CMake 设定openssl 源码文件目录 ;
    • Xcode build .a
    • android studio build .so
