pack Transfer Binary Data

  1. store the send data into a binary array which size is large 4bytes.
  2. store the needed parse type into the up binary array’s last 4bytes
  3. send the new packed binary data to the other.

upack Transfer Binary Data

  1. Receive the entire binary data to a binary array.
  2. Extract last fixed 4bytes data to get the parse response type;
  3. According the type’s value ,client can parse the protobuf to proper client language object.
