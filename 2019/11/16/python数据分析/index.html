##前言

现在网络上充斥着各种学习python的推广或者培续班,甚至上升到朋友圈,都出现学python的广告,几乎是人人都被覆盖过,更夸张的是,居然潘石屹也发微博进行公告天下,他也要学python,或许我们曾经或者现在干it行业,学过或者正在学python,以为学会python好找工作,尤其是去找数据分析的工作，然鹅学会了python却还是不知道该怎么做数据分析，久而久之就只记得当初为啥学python ,却不知道学了python能干嘛.

数据分析的工具有哪些?

工欲善其事,必先利其器。在了解了数据分析的工作内容之后，我们应对着有啥工具可以进行数据分析的工作呢？

就我了解的同事有用以下工具

  1. 微软的Excel 和 POWER BI;
  2. 用Pycharm结合python各种库进行开发的;
  3. 用Zeppelin 进行sql和python以及spark组合开发;
  4. 用jupyter;
  5. 用Conda ,有兴趣的可以自己去试试看,也是全家桶
  6. …等

了解过这些工具之后，折中方案是选用 jupyter,主要是因为以下几点

  1. 对比Excel 和 Power BI ,无需过多的Excel技巧掌握, 降低学习门槛;
  2. 对比Pycharm 和 Python开发,无需将代码和文档分离;
  3. 对比zeppelin 的话，也是对初学者降低门槛，当技能步入熟练阶段,有能力搭建zeppelin或者公司有这环境的话，也还是可以试试看的;
  4. Conda 的话，软件全家桶,包含着jupyter,有兴趣可以尝试下;
  5. Jupyter 的代码文件可以共享,其他小伙伴只要配置了jupyter 环境,就可以拿到你的文档文件,直接在jupyter 正常运行，无需额外的各种工作,这极大的方便了与你一起工作的小伙伴。当然你从github 里找到的学习资源,也绝大部分是jupyter 形式的，

综上:我建议想学习了解数据分析的小伙伴，从jupyter 开始;
