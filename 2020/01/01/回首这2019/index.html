回首这2019

发表于 2020-01-01 | 分类于 杂谈
字数统计: 1,250 字 | 阅读时长 ≈ 4 分钟

这一年

雄关漫道真如铁，而今迈步从头越

行业西沉

从18年开始,每到年底,网上充斥着各种论调,说 2018(2019)年 经济大环境是过去10年最坏的一年,也是未来10年最好的一年。

伴随着这种论调出现的现象是,曾经的互联网行业扛把子、独角兽、明星创业公司,在网上频繁被曝光,采取各种花式小动作,干的就是一件事,就是开源节流**(裁员)**,对这些现象,我们不是旁观者,我们身在其中,感同身受。

这些现象,像极了刚毕业那会儿,由于iOS、Android 等移动互联网技术的发展,改变了社会上工作角色的分工,貌似大家一窝蜂的都涌向了无线互联这个行业方向。

从公司层面上说,有当记者的跨界创建了摩拜、华尔街见闻等明星公司;
有腾讯、阿里等传统互联网公司拥抱无线技术的革新,投入移动端的开发,有了国民 App (微信/支付宝等)。
还有大学生创建的饿了么、ofo 等O2O公司

从个人就业层面上说,由于以iOS 和android 系统构建的移动应用生态系统,满足了人们日益增长的业余娱乐需求和传统功能手机的应用单一性的矛盾；

导致传统的软件开发或者嵌入式开发从业者要么失业,要么转行,尤其明显的是以Nokia的手机应用开发者为著,这批人在2010年以前,那可是外企的IT门面。然而2011年以后,塞班开发者基本上绝迹了,我想他们当时也是各种花式被优化吧。

个人总结

当先进的生产力出现的时候,需要配套的生产关系,生产资料;
如果没有,我们要拥抱这种变化,尝试着创造它。
否则,要么死扛(累),要么淘汰。

换言之,就是当有高收益行业或高薪职业出现的时候,人们就会逐利,会去适应它,否则就是在等待淘汰。

早起的鸟儿有虫吃

感恩

2010年暑假实习,感恩让我碰到一个见过世面的好mentor;

他先让我踏入了移动开发这个行业,从事开发iOS 和 Android 的工作，并推荐我附加学习一门脚本语言Python,我还记得当时的理由是他在美国的学数据挖掘的同学都在学这个语言,数据分析和信息挖掘是趋势。

从 C++ 切换到 Objective-C / Python,让我见识到了移动互联网的发展历程,见证了一些互联网公司的崛起与没落。也真正意识到了数据分析和数据挖掘在移动互联网生态链中的分量。Python的业余开发方向从 web 向数据分析方向倾斜。

职业转变

从深圳来到上海 ,不光是工作地点的改变,也有工作方向的改变;

在和mentor创业的过程中,第一次全流程参与到软硬件产品从概念到原型,从原型到设计、实现、线上部署、线下发布;
偶尔会和海外行销部门进行运营分析决策的讨论。

接受这项工作的内容和面对这项工作的性质,我需要作出角色的改变,从一个单纯的 mobiler 到 data analyst,需要从最初的只负责移动开发到数据开发、再到算法开发。才能cover住我的工作内容,保留mentor的信任。

如今,虽然已经换了东家,但过往的经历对现在的工作显得弥足珍贵,也正好赶上了算法和数据的工作的**“春天”**。

心得体会

从移动到数据角色的改变,给我的感觉就是下面这首诗

转业之初对数据算法的莫名胆怯;
数据开发的过程中,了解原理和工程实践,效果提升之后,面对算法也不那么惧怕了;
即使真的后面10年大环境不好,社会上也会有新的行业机遇在等着有准备的我们;
我们无需过度焦虑、过度惊慌

展望

面对年底裁员和经济衰退这些宏观观上不可控的因素,我们需要做的事就是保证微观上的自我追求不停止,面对陌生的变化,积极拥抱它。

在这里,引用习大大的2020年新年寄语共勉

只争朝夕,不负韶华!!!
